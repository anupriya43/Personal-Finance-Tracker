# Personal Finance Tracker

A **Python-based Personal Finance Tracker** that helps users record, manage, and analyze their daily income and expenses.  
The application allows users to categorize transactions, calculate total balance, and store financial data for future reference.

---

## üìå Project Description

Managing personal finances manually can be difficult and error-prone.  
This project was developed to provide a **simple and efficient way to track income and expenses**, helping users understand their spending habits and maintain better financial control.

The application is **menu-driven**, easy to use, and stores data persistently using file handling so that records are not lost after execution.
